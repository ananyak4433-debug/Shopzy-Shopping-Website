# Shopzy Shopping Website

This project is a responsive e-commerce shopping website built with React and Redux. Users can browse products, view product details, and add items to a cart. Redux manages global state such as product listings and cart data. The app features dynamic product rendering, smooth navigation, and a scalable component-based architecture.

## Features
- Product listing
- Product details page
- Add to cart functionality
- Global state management with Redux
- Responsive UI

## Installation

Clone the repository:

npm install

Run the project:

npm start

Open http://localhost:3000 to view it in your browser.

## Build

To create a production build:

npm run build
